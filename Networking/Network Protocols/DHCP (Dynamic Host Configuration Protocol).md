**DHCP manages and assigns IP addresses**
- DHCP is dynamic, because most devices do not have fixed IP addresses.

**DHCP operates on Layer 7 - Application of [[OSI Model]]**
- Utilizes two (2) UDP Ports
	- Server -- Port 67
	- Client -- Port 68
